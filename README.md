## Instructions:

### Ignore this If you already installed Flask library in Python

1. Locate to the folder that contains "Python.exe" (you can use the command "where python" in CMD to find it)
2. Open your command prompt (CMD) in the folder then type:
- pip install Flask
3. After It's completed download, you may begin following the steps below

### How to run

1. Open your command prompt (CMD) in the project

2. Run one of the following commands:
- python app.py
- py app.py

3. Open a web browser and navigate to: http://localhost:5000/
